Code a Page - HTML

Web pages are coded using HTML (Hypertext Markup Language). The codes consists of tags in angle brackets < >. Some tags can contain contents including further tags. The HTML tags control how all web browsers draw your page.

A web page file usually starts with a DOCTYPE tag that lets the browser know the format of the rest of the file.

The rest of the page is then wrapped in a <html> tag. Notice how the closing tag, at the bottom of the code, contains an extra / to indicate the end of it's counterpart above.

The 'tags' we code are interpreted by browsers to form HTML 'Elements' and generally two terms can be used interchangably.

Other HTML elements on this page include:

  • <head>...</head> - This is a place where metadata about your page can be specified. Usually these tags are not rendered directly on your page.
  • <body>...</body> - This is where you put the main content of the web page.
  • <h1>text</h1> - A top level heading
  • <p>text</p> - Paragraphs

Try some other tags

Here are some more to try:

  • <b>text</b> - You can make things <b>bold</b> by wrapping start and end tags around sections of text.
  • <i>text</i> - This one works the same as bold only for <i>italics</i>.
  • <input> - This tag provides that you might use to later capture some information from the user of your page. Input tags do not need an ending tag.
  • <img src=""> - Images can be inserted into your page. Some tags need additional attributes set in order to work. For images the src attribute (aka source) needs a URL so that the browser know where to get the image from. You can get a URL for an image from the internet or here is one we have prepared for you. <img src="https://code.qantas.com/learn-to-code/images/cat.jpg">
  • <a href="">link text</a> - The Anchor tag lets you create a link from your page to any other location on the internet by setting the attribute href (aka hypertext reference) to a URL. The content you put between the start and end tags becomes <a href="https://en.wikipedia.org/wiki/Clickable">clickable</a> and the browser does the rest.
